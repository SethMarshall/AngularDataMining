This web application is a tool for visualizing k-means clustering, a common but fundamental algorithm in data mining. Informally, k-means works as follows:

Choose a number of clusters. We then attempt to group data points such that our clustering is as "tight" as possible. That is, the distance from each point to its cluster center is minimized.

A bit more technically, k-means minimizes the sum of the squared distance between every point and its current centroid. If an alternate cluster center is closer a point is reassigned to that new center. Reassigning a point to a new cluster, however, alters a cluster's average, and thus a racalibrates the cluster center. Distance here can also be thought of as the sum of sqaured error (SSE), the output of which is displayed in the data panel of the home page.

The demo simplifies K-means insofar as the data sets are generated randomly on every run; the number of clusters is set to four, and data points are restricted to two dimensions. The latter in particular is necessary to provide a user-friendly real-time visualization. In principle, of course, the number of clusters is variable and the dimensions to a data set unlimited.
